## 平台支持状态

❌ **不支持Mac平台**

v2.0版本不支持Mac平台。如需Mac支持，请使用v2.1+版本。

### 升级建议
```bash
# 使用v2.1版本获得完整Mac支持
cd ../2.1/mac/
./setup_mac.sh
```
