# Tier 1 Micro 策略 (1千-1万元)

## 档位特征

- **资金规模**: 1,000 - 10,000 元
- **Arena测试**: 零约束，完全自由进化
- **滑点基线**: 0.1% - 0.2%
- **流动性要求**: 低

## 策略特点

在这个档位，策略可以：
- 高频交易（资金小，灵活性高）
- 集中持仓（单股可达20%+，由Arena进化决定）
- 快速进出（滑点影响小）
- 探索性策略（试错成本低）

## Arena进化参数示例

```python
{
    'max_position': 0.95,        # 最大总仓位95%
    'max_single_stock': 0.25,    # 单股最大25%
    'max_industry': 0.60,        # 单行业最大60%
    'stop_loss_pct': -0.08,      # 止损-8%
    'take_profit_pct': 0.15,     # 止盈15%
    'trading_frequency': 'high', # 高频交易
    'holding_period_days': 2     # 持仓2天
}
```

## 注意事项

1. **参数由Arena进化**：上述参数仅为示例，实际参数由Arena测试中策略自由进化得出
2. **无预设约束**：不预设任何交易模式（刺客/狼群/利维坦）
3. **滑点模拟**：Arena测试时不模拟滑点（Tier1-2滑点影响极小）
4. **Z2H认证**：必须通过完整验证流程才能进入实盘
